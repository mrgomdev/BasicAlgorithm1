   1 , ( 1 + 2 ) , ( 1 + 2 + 3 ) ,.... , ( 1 + 2 + .. + n) ...
  1 항    2 항          3 항                 n 항
n 항 까지의 합을 구하는 프로그램을 작성하시오.
1 + ( 1 + 2 ) + ( 1 + 2 + 3 ) ,.... + ( 1 + 2 + .. + n)
입력

n 은 100 이하의 자연수이다.
출력

합을 출력한다.
입출력 예

입력

3

출력

10
