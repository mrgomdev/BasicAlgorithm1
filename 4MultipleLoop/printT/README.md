프로그램 명: printT
제한시간: 1 초
홀수 n 을 입력으로 받아 T 를 출력하는 프로그램이다.

입력

홀수 n 을 입력받는다. 3 <= n <= 20
출력

출력 예의 형식으로 출력한다. 불 필요한 공백을 출력해서는 안된다.
입출력 예

입력

5

출력

*****
  *
  *
  *
  *
출력 보충

공백이 . 이라면
*****
..*
..*
..*
..*
[질/답] [제출 현황] [푼 후(2)]
[ 채 점 ] [홈으로]  [뒤 로]
