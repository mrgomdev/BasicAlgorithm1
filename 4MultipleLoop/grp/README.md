프로그램 명: grp
제한시간: 1 초
1에서 n까지의 자연수를 다음과 같이 연속한 k개의 수로 차례로 묶어 놓았다.

(1,2,3...k), (2,3,4..,k+1),...,(n-k+1,.., n-1, n)

이 때, k 개의 수의 합이 15 의 배수인 것은 모두 몇 묶음인지 구하여라.

입력

n , k(n>k) 가 입력으로 주어진다. n 은 1000 이하의 자연수이다.
출력

묶음이 15 의 배수인 것의 개수를 출력한다.
입출력 예

입력

100 3

출력

19
